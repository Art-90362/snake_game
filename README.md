# 🐍 Snake Game Installer

## Установка:

1. Загрузите `SnakeGame_Installer.exe` на ваш ПК
2. Запустите от имени **Администратора** (правой кнопкой → "Запустить от имени администратора")
3. Следуйте инструкциям
4. Игра будет установлена в `C:\Program Files\SnakeGame\`

## Удаление:

- Зайдите в `C:\Program Files\SnakeGame\` и запустите `uninstall.bat`
- Или удалите через "Панель управления → Программы и компоненты"

## Файлы после установки:

- `snake_game.exe` — основной файл игры
- `SnakeGame.bat` — быстрый запуск
- `uninstall.bat` — скрипт удаления
